# Project Context

## What This Repository Is

- Repository: `agam-lang/agam-intellij`
- Purpose: Provide IntelliJ/IDEA integration for Agam development with syntax highlighting, project support, and eventual LSP connection.
- Technology: Kotlin/Java + IntelliJ Platform SDK

## Organization Context

- This is part of the `agam-lang` ecosystem.
- The core Agam compiler lives at `agam-lang/agam`.
- The standard library lives at `agam-lang/std`.
- The scientific platform lives at `agam-lang/agamlab`.

## Key Areas

- Syntax Highlighting
- Project Support
- LSP Integration
